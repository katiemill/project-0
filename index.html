The World Wide Web is a collection of HTML documents. HTML = HyperText Markup Language. The WWW was invented in the 1990s and has 30 billion pages. Files found on the Web can be plain text, HTML, images, videos, and music.

Major Pieces:
Your browser: When you go to a website, your browser makes request via internet to servers using protocol called http.
Server: Responds with files that your browser displays. Servers sit and host files.

HTML:

Text content/what you see
Markup/what it looks like/how it's arranged
References to other documents (images/videos)
Links to other pages

Mostly, plain text is just plain text in HTML

How to add a hyperlink: < a href="website name"> content< /a> with no spaces
a is anchor tag, href is an attribute name
HTML Markup: < name> contents < /name> with no spaces, where opening tag plus contents plus closing tag equals element
Exception: Void tags have no content and no end tag. Such as < br> and the tag to add an image: < img src="url" alt="text">
src is source, alt refers to "alternate," or text that displays when image doesn't load

Inline vs block
< br> is simply a line break, so it is inline, as are < b>, < em>, and < img>.
< p> is block--makes invisible block around text, sections it off. Is not a void tag--requires content and end tag

container elements--elements that contain stuff inside them (more on that later?)

Control how text looks: span/div
< span>content< /span> is inline
< span class="foo"> text < /span> turns 2 lines into one?

< div>content< /div> is block, as are < p> and < form>
